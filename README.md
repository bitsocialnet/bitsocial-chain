# Bitsocial Network

This repository is reserved for the future Bitsocial Network implementation.

Bitsocial Network is the proposed L2/appchain economic layer for Bitsocial apps. It is planned to power practical network features such as decentralized `.bso` names, awards, tipping, shared liquidity, monetization, and other financial primitives that make Bitsocial apps harder to starve or capture.

There is no node software, chain implementation, contract system, or deployment tooling here yet.

For now, the canonical public context lives in:

- [Bitsocial Network docs](https://bitsocial.net/docs/bitsocial-network/)
- [Phase 2 of the Bitsocial master plan](https://bitsocial.net/#master-plan-phase-2)
